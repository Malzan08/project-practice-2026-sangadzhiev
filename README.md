## Часть по выбору

В качестве задания части по выбору (согласно п. 2.2 ТЗ) была выбрана тема: **«Разработка легковесного многопоточного веб-сервера (HTTP-сервера) с нуля на Python»**.
В рамках задания реализованы:
* Многопоточная обработка TCP-соединений с использованием пула потоков (ThreadPool).
* Префиксный маршрутизатор (Trie Router) для динамических путей и статических файлов.
* Умное кэширование на базе хэш-сумм файлов (ETag, Last-Modified) и сжатие Gzip.
* Поддержка частичной отдачи данных (Range Requests) для потокового воспроизведения медиа.
* REST API для интеграции с базой данных социологического опроса студентов.
