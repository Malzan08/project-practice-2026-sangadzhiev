# Полезные ресурсы и информационные источники

В данном документе приведен список ресурсов, литературы и веб-сервисов, использованных в ходе выполнения проекта и написания отчетов.

---

## 1. Официальные ресурсы организации и конференции
*   [Официальный сайт Московского Политеха](https://mospolytech.ru/) — информация о структуре университета, направлениях подготовки и правилах прохождения учебной практики.
*   [Портал СДО (LMS) Московского Политеха](https://online.mospolytech.ru/) — платформа для загрузки отчетной документации и прохождения учебных курсов.
*   [Официальный сайт конференции DataFest 2026](https://datafest.ru/) — программа докладов, секции по анализу данных, машинному обучению и NLP-анализу.
*   [Карьерный марафон Московского Политеха](https://mospolycareermarafon.ru/april2026) — сайт карьерного мероприятия для студентов.

## 2. Разработка и программирование (HTTP-сервер и Сайт)
*   [Python Socket Programming (Real Python)](https://realpython.com/python-sockets/) — подробное руководство по программированию сетевых сокетов в Python.
*   [Официальная документация Python: модуль socket](https://docs.python.org/3/library/socket.html) — спецификация низкоуровневых сетевых интерфейсов.
*   [MDN Web Docs: HTTP-протокол](https://developer.mozilla.org/ru/docs/Web/HTTP) — исчерпывающее руководство по структуре HTTP-запросов, ответов, заголовков и статус-кодов.
*   [Документация по CSS Grid и Flexbox (Doka)](https://doka.guide/css/) — справочные материалы по современной адаптивной верстке.
*   [Официальный сайт библиотеки Chart.js](https://www.chartjs.org/) — документация по созданию интерактивных графиков и дашбордов.

## 3. Социология и анализ данных
*   [Методология социологических исследований (Stepik)](https://stepik.org/course/100) — открытые лекции по разработке опросов и выборочному методу.
*   [Шкала Лайкерта в исследованиях образования](https://www.researchgate.net/) — научные публикации о применении порядковых шкал для оценки качества преподавания.
*   [Прикладная статистика на Python (SciPy)](https://scipy-lectures.org/) — руководства по вычислению корреляций, дисперсии и статистической обработке данных опросов.
*   [Курс по NLP от Hugging Face](https://huggingface.co/learn/nlp-course) — учебные материалы по токенизации, классификации текстов и анализу тональности (Sentiment Analysis).
