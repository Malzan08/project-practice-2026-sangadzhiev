# Участники проекта и их вклад

В реализации проекта «Социологический мониторинг и анализ оценки эффективности ППС» принимала участие группа из 6 человек. Каждый участник выполнял свои задачи в соответствии с утвержденным планом работ.

---

## 1. Белов Артём Дмитриевич
*   **Роль:** Руководитель проекта, Backend-разработчик (вымышленный участник).
*   **Личный вклад:**
    *   Общее руководство командой, распределение задач и контроль контрольных сроков.
    *   Разработка архитектуры системы и проектирование серверной логики.
    *   Создание многопоточного HTTP-сервера с нуля на Python с использованием низкоуровневых сокетов (без фреймворков).
    *   Реализация маршрутизатора запросов на основе префиксного дерева (Trie Router).
    *   Разработка системы сессий, механизмов авторизации пользователей и middleware для логирования запросов.

## 2. Смирнова Дарья Алексеевна
*   **Роль:** Frontend-разработчик, веб-дизайнер (вымышленный участник).
*   **Личный вклад:**
    *   Разработка визуального стиля проекта, концепции Glassmorphism и цветовой палитры интерфейса.
    *   Верстка адаптивного веб-интерфейса сайта (HTML5/CSS3) для всех типов устройств (мобильные телефоны, планшеты, десктопы).
    *   Интеграция интерактивных графиков и диаграмм (Chart.js) в аналитический дашборд.
    *   Создание микро-анимаций интерфейса и переходов между страницами.
    *   Оптимизация производительности рендеринга статических страниц сайта.

## 3. Сангаджиев Малзан Гаряевич
*   **Роль:** Аналитик данных, социолог (реальный участник, студент группы 251-951).
*   **Личный вклад:**
    *   Разработка методологической основы мониторинга: выбор критериев оценки ППС и адаптация шкалы Лайкерта.
    *   Составление и верификация структуры анкеты для проведения опроса студентов.
    *   Проведение разведочного анализа данных (EDA) по собранным ответам студентов, расчет средних значений и дисперсии оценок.
    *   Реализация алгоритма корреляционного анализа Пирсона для оценки связи успеваемости студентов и оценок преподавателей.
    *   Подготовка текстовых материалов, документации проекта в формате Markdown и итоговых отчетов о прохождении практики.

## 4. Козлов Михаил Игоревич
*   **Роль:** Data Engineer, администратор базы данных (вымышленный участник).
*   **Личный вклад:**
    *   Разработка схемы реляционной базы данных для хранения информации о преподавателях, группах, студентах и анкетах.
    *   Оптимизация SQL-запросов для быстрой выборки статистических данных для дашборда.
    *   Организация процесса очистки данных от аномалий (data cleaning) и фильтрации спам-ответов.
    *   Создание ETL-процессов для импорта списков ППС из внешних систем вуза.

## 5. Морозова Анастасия Сергеевна
*   **Роль:** Социолог, разработчик анкет (вымышленный участник).
*   **Личный вклад:**
    *   Проведение качественных исследований (фокус-группы со студентами) для выявления наиболее значимых критериев оценки ППС.
    *   Разработка инструкций для респондентов и организация процесса анкетирования в учебных группах.
    *   Контроль репрезентативности выборки при проведении мониторинга.
    *   Тематическое кодирование текстовых комментариев студентов о качестве преподавания.

## 6. Васильев Дмитрий Петрович
*   **Роль:** QA-инженер, тестировщик (вымышленный участник).
*   **Личный вклад:**
    *   Разработка тестовых сценариев и проведение функционального тестирования клиентской и серверной частей.
    *   Написание модульных тестов (`unittest`) для проверки парсинга HTTP-запросов и ответов сервера.
    *   Проведение нагрузочного тестирования HTTP-сервера с использованием утилит ApacheBenchmark (ab).
    *   Поиск и документирование багов верстки сайта на различных разрешениях экранов.
